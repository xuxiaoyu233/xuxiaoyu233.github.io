初学PHP的心得体会总结（1）
Created2020-10-31|Updated2020-10-31

1.开头不同

在C语言中需要使用预处理（如#include(stdio.h)等)

在PHP中以 ‘‘结尾

2.函数定义时有区别

在C语言中要先确定变量的类型（int,float,char等）再赋值

在PHP中变量要以’$‘开头不需要说明变量类型

3输出时函数有区别

在C语言中用printf()作为输出语句

在PHP常用“echo”输出，但也可以用print输出（此输出有限制）。

4.换行符不同

在C语言中用printf(“\n”)表示换行，或者一些函数自带换行 。

在PHP中使用<br表示回车换行一次

为回车换行两次。

5.运算符有区别

在PHP中存在特殊运算符如

1.“===”

在PHP中 5==”5“的结果是ture。但5===”5“的结果是false。

因为”===“的意思表示不仅需要两个变量的值相等而且需要它们的类型相同

2.”<>“

4.<=> 称为太空船

该组合比较运算符可以轻松实现两个变量的比较，当然不仅限于数值类数据的比较。

eg: //注意（推测）：当数字与字符比较时，除去零以外的数总大于字符，故输出1.
$c = $a <=> $b; //类似：当负数与字符比较时总小于字符
解析如下：

如果 $a > $b, 则 $c 的值为 1。
如果 $a == $b, 则 $c 的值为 0。
如果 $a < $b, 则 $c 的值为 -1。

适用于整数，小数和字符。
